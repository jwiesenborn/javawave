SYNTHESIS

Synthesis

Post by Jesse, 28 April 2019

I fused Dr V's musicXML file together with my math synthesizer, and you can download the result here: A Sigh (16-bit). You may need software that can play OGG files such as VLC media player, but it might work if you just click the link.

The program is something of a gargoyle. Web browsers really aren't optimized for data processing, and will struggle with the millions of logic operations needed to handle uncompressed sound. It crashed my computer multiple times. I ended up computing each of the seven pieces individually, and then I compiled them with Audacity.

This is only the accompaniment part. It sounds kind of like an organ, and I think it turned out awesome. Sing along if you know the words!
